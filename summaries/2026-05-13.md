# llama.cpp daily summary — 2026-05-13

A highly active day spanning GPU backends, multimodal improvements, and a major new evaluation framework. GPU work dominated, with significant commits landing for WebGPU, Vulkan, Metal, OpenCL, and Qualcomm Hexagon simultaneously. The standout addition is `llama-eval`, a full Python benchmark harness supporting AIME, GSM8K, GPQA, and more.

## CUDA / GPU (General)

- **OpenCL**: Add opt-in Adreno xmem F16×F32 GEMM kernel for prefill on Qualcomm Adreno GPUs — notable perf path for mobile inference — #22755 `a9883db`
- **Hexagon**: Eliminate scalar VTCM loads by routing splat operations through new HVX `hvx_vec_repl` helpers; also optimises per-group scale handling in HMX matmul — #22993 `856c3ad`
