# llama.cpp daily summary — 2026-04-30

Active day with 14 commits across GPU backends (CUDA, WebGPU, Hexagon), CPU
platforms (ARM SVE, RISC-V, AIX), speculative decoding, and the web UI.
Notable highlights include a CUDA kernel fusion for Mamba/SSM models and a
tensor-parallelism correctness fix.

## CUDA / GPU

- **CUDA: fuse SSM_CONV + ADD(bias) + SILU** — reduces kernel launch overhead
  for Mamba-style SSM models; new backend-ops tests included.
  (#22478, `098705a`)
- **ggml-cuda: refactor fusion code** — large internal reorganisation of
  CUDA kernel-fusion dispatch; no user-visible behaviour change.
  (#22468, `3142f1d`)

## WebGPU

- **Fix FlashAttention support check** — corrects a bug where devices lacking
  subgroup support would incorrectly enable FA; also falls back when the KV
  tile doesn't fit. (#22492, `d6a5094`)

## Hexagon / Qualcomm

- **hexagon: make vmem and buffer-size configurable** — bumps default vmem to
  3.2 GB, adds auto-detection of available vmem, and moves pinned mmap
  management to the host. (#22487, `41a63be`)

## CPU / ARM

- **ARM SVE tuned kernel for `gemm_q8_0_4x8_q8_0()`** — hand-optimised SVE
  path that should improve throughput on Fujitsu A64FX and similar cores.
  (#21916, `bdc9c74`)
- **Disable tiled matmul on AIX** — `vec_xst` near 4 KB page boundaries
  causes segfaults; falls back to mnpack on AIX. (#22293, `1cbc846`)

## CPU / RISC-V

- **SpacemiT IME build fix** — appends `_xsmtvdotii` to `-march` when
  `GGML_CPU_RISCV64_SPACEMIT=ON` so the IME inline-asm path assembles
  correctly. (#22317, `b5c4227`)

## Speculative Decoding

- **Discard last drafted token with low probability** — avoids accepting a
  low-confidence tail token that can degrade output quality.
  (#22506, `683c5ac`)

## Common / Sampling

- **Do not pass prompt tokens to reasoning-budget sampler** — prevents
  prompt tokens from polluting reasoning-budget accounting.
  (#22488, `d775992`)

## Web UI

- **Fix slow mic stop and WAV encode** — instant microphone teardown,
  race-free recorder restart, and faster PCM→WAV encoding via hoisted
  channels. (#22480, `59237bf`)

## Fixes

- **Leak logger instance on Windows** — intentional singleton leak prevents
  a hang/crash at process exit when the logger thread outlives static
  destructors. (#22273, `7b95ea5`)
- **TP: fix delayed AllReduce + zero-sized slices** — tensor-parallelism
  correctness fix for edge cases with empty slices. (#22489, `739393b`)

## Misc

- ggml version bumped to 0.10.1; ggml submodule synced. (`4b221b7`, `b1d5f5b`)
